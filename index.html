Yahoo Finance는 모바일 브라우저에서 직접 조회가 차단됩니다.
Cloudflare Worker를 무료로 설정하면 실시간 주가를 받을 수 있습니다.

설정 방법:
1. dash.cloudflare.com → 무료 계정 생성
2. Workers & Pages → Create → Hello World
3. GitHub 저장소의 cf-worker.js 코드 붙여넣기
4. Deploy 후 생성된 URL 아래에 입력
